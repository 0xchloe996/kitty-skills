# Remotion Best Practices

This skill provides domain-specific knowledge for working with Remotion, a React-based framework for video creation and animation.

## Overview

The skill encompasses best practices across multiple aspects of video creation in Remotion, from fundamental animations to advanced features like 3D rendering and dynamic metadata calculation.

## Key Topic Areas

The skill covers:

- **Core Animation**: Animations, timing, transitions, and text animation patterns
- **Media Assets**: Images, videos, audio files, GIFs, and font loading
- **Data & Visualization**: Charts, data visualization, and Lottie animations
- **Advanced Features**: 3D content with Three.js, transparent video rendering, and map integration with Mapbox
- **Utilities**: Text and DOM measurement, sequencing patterns, parameter schemas, and video format detection
- **Styling**: TailwindCSS integration and caption/subtitle handling

## Usage

Consult individual rule files in the `./rules/` directory for detailed explanations, code examples, and implementation patterns specific to each topic area.

Each rule file provides focused, technical guidance on its respective domain within the Remotion ecosystem.
